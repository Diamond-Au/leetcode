## 基本的位运算知识

> n的二进制表示中第K位是几

```c++
// n >> k & 1
```



> lowbit(x)  

```c++
// x & -x 
```

> 应用

```c++
#include <iostream>
using namespace std;


int lowbit(int x) {
    return x & -x;
}

int main() {
    int n;
    scanf("%d", &n);
    int res = 0;
    while(n) {
        n = n - lowbit(n);
        res ++:
    }
   
    return res;
}
```
